## mpt_hw3
В этом репозитории находится выполненное домашнее задание по теме "Парсин, Git и тестирование на Python".

Выполнил Рыжков Василий.

## Цель
Разработать систему для автоматического сбора данных о книгах с сайта [Books to Scrape](http://books.toscrape.com). Реализовать функции для парсинга всех страниц сайта, извлечения информации о книгах, автоматического ежедневного запуска задачи и сохранения результата. Оформить репозиторий на GitHub c Readme, артефактами и юнит-тестами.

## Инструкции по запуску
Создайте виртуальную среду для Python 3.10,
и установите все зависимости из файла requirements.txt

Все необходимое для проверки задания можно найти и запустить в ноутбуке, расположенном в notebooks\HW_03_python_ds_2025.ipynb

При необходимости можно запустить юнит-тесты отдельно:
```
cd ./tests/
pytest test_scraper.py
```
